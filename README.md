# Стальро — сайт ИП Хрюкин Е.А. на Node/Express

## Запуск
1. Установите Node.js LTS.
2. В терминале:
   - cd C:\Project
   - npm install
   - npm run dev (или npm start)
3. Откройте http://localhost:3000

## Страницы
- Главная: / — с калькулятором стоимости
- Каталог: /catalog.html — с категориями и фильтрами
- О компании: /about.html
- Админ-панель: /admin.html — управление заявками

## Функционал

### Заявки на обратный звонок
- POST /api/callback — сохранение заявок с возможностью загрузки файлов
- Поддержка полей: имя, телефон, email, тип продукции, комментарий, файл
- Заявки сохраняются в файл data/callbacks.json
- Загруженные файлы сохраняются в data/uploads/

### Админ-панель
- GET /api/callbacks — получение всех заявок
- PUT /api/callbacks/:id — обновление статуса заявки
- DELETE /api/callbacks/:id — удаление заявки
- GET /api/callbacks/export/csv — экспорт заявок в CSV

### Калькулятор стоимости
- POST /api/calculate — расчёт стоимости металлоконструкций
- Параметры: вес, сложность, покрытие

### Каталог
- Категории: Балки, Колонны, Фермы, Лестницы, Ограждения, Нестандартные
- Поиск по названию
- Фильтрация по категориям
- Автозаполнение типа продукции в форме заявки

## Структура проекта
- `server.js` — Express сервер с API
- `public/` — статические файлы (HTML, CSS, JS)
- `data/` — данные (callbacks.json, uploads/)
- `package.json` — зависимости (express, multer)
